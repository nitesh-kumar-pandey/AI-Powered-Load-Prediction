About

About This Project

A full-stack machine learning web application built as a learning project.

LoanSense is a loan approval prediction system powered by a trained Artificial Neural Network. It was built to demonstrate how machine learning models can be integrated into real web applications to provide actionable financial insights.

The backend is a Flask application that loads a pre-trained Keras / TensorFlow ANN model and exposes it via a simple POST endpoint. Input features are normalized using the same StandardScaler fitted during training before being passed to the model. Every prediction is persisted to a SQLite database using Flask-SQLAlchemy, creating a full audit trail of all applications.

The model was trained on real-world loan dataset features including credit score, income, asset valuations, loan term, and personal profile data — giving it a well-rounded view of applicant eligibility.
